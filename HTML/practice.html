HTML

HTML stands for hyper text markup language which was invented by tim berners lie in the year of 1995 . HTML is the basic building block to create web pages nd to give structure for web pages.

Why HTML?

Markup Language

The language that represents data on web page using markup tags that language is called Markup language.
